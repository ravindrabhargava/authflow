You can use Postman to create user as follows:
- Create user method
* POST localhost:3600/users
 Body: {"firstName" : "ABZ",
"lastName" : "XYC",
"email" : "test@gmail.com",
"password" : "password01"}

Response will generate a user id, something like this:
{
    "id": "5f13c55c46b5dd00110d7837"
}
